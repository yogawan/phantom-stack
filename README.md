# PHANTOM Stack v2

Fullstack Next.js ecosystem for building scalable SaaS and cinematic web apps.

## What is PHANTOM?

PHANTOM = Performance, Headless, Animation, Next, Tailwind, Observable State, Maintainability

Modern fullstack stack focused on:
- Performance
- Scalable Design System
- Cinematic UX
- Maintainable Architecture
